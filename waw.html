Sales assist portal

Current - 3+ months
Description:

Being a long running project, the project has an extensive design system, design libraries and a strong visual design language. The primary task of any new designer joining the design team is to get aquainted with the above.

Data Representation Model

5 months
Description:
The client is a South Korean electronic giant who came to us with the project. The scope wasdesign and development of over 15 web-widgets and dashboards. Each of these widgets were unique in requirements and were designed to be used individually and/or in a composition, as required. The common thread to each of these widgets was the Design system and visual design.

Process:
Given the volume of work, the 15+ widgets were segemented into sprints across months. The deign of each widget went through a rigourous process of research, stakeholder’s feedback, design presentations and iterations, before being taken up for development.

Challenges and Solutions:
  • The client communcated in their native tongue only . We bridged the lingual/cultural gap by using translators and visual mediums.
  • The delivery demand was high. We used the existing design system to optimise the design delivery.

My Responsibilities:
  • Client communication
  • UX Research and Design

Dashboard design for a Product in Logistics Domain

1 month
Description:
The project is a web platform which focuses on optimizing logistics by compiling and highlighting bottlenecked areas and suggest best alternatives to resolve the issues.

Process:
  • The Persona of a planner and their user journey were the core while designing.
  • Every screen and interactions were designed while keeping in mind the affordance and the ease of access for this user category.
  • As this journey was enacted over and over with the team, more insights came up on how to improve the UI of the dashboard.
  • The dashboard would enable the user to perform the most essential tasks within as few steps as possible.
